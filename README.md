## Descripción del Proyecto
Aplicación web responsiva para explorar películas y series usando la API de TMDB.

## Características
- Visualización de películas/series populares
- Búsqueda por título y categoría
- Diseño 100% responsivo con Bootstrap
- Páginas de detalle completas

## Tecnologías Utilizadas
- Python 3.x
- Flask
- Bootstrap 5
- Jinja2
- API TMDB
